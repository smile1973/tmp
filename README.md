# Game Introduction

- 彈幕動作遊戲(~~絕本練習器~~)
- 閃躲彈幕 攻擊敵人
- 上下左右控制方向 ZXCV技能
- rogue like元素 擊敗敵人獲得加強
- 影片連結: https://www.youtube.com/watch?v=A3rulL2XkyU&pp=ygUVcmFiYml0IGFuZCBzdGVlbCBzb2xv
- 目標實作1個腳色 至少兩個敵人
    - 一個腳色包含四個技能，受擊時扣一格血，無敵時間1秒
    - 每個敵人包含2~3個攻擊pattern
